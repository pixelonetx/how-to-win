# HowToWin

Windows 使用与开发笔记。

本文档面向日常使用 Windows，并希望在 Windows 上整理开发环境的读者。

在线阅读：

```text
https://pixelonetx.github.io/how-to-win/
```

## 本地开发

安装依赖：

```powershell
npm install
```

启动本地开发服务器：

```powershell
npm run docs:dev
```

默认访问：

```text
http://localhost:5173/
```

构建静态文件：

```powershell
npm run docs:build
```

本地预览构建结果：

```powershell
npm run docs:preview
```
