# 安装
以华为 Windows 笔记本为例

## 0. 需要用到的工具

一个容量不小于 8G 的 U 盘，并将 U 盘格式化为 FAT32 格式。
操作步骤：右键点击 U 盘，选择格式化，将文件系统选择为 FAT32（默认），点击开始。



一台可以连接 U 盘，可以上网的计算机，用来下载系统安装盘。
需要重装系统的计算机。

为了避免数据丢失，请您提前做好全盘数据备份。

## 1. 制作系统安装盘

将准备好的 U 盘插入联网的计算机中，打开微软官方网站 [Windows11](https://www.microsoft.com/zh-cn/software-download/windows11)，在下载提示中点击保存，下载系统安装盘制作工具 MediaCreationTool.exe。




在下载完成后，点击运行（也可以在下载路径下找到 MediaCreationTool.exe，双击运行）。
出现适用的声明和许可条款时，用户阅读后若同意则点击接受。


选择语言和版本：对这台电脑使用推荐的选项默认勾选，即会根据您当前使用的电脑自动选择最合适的Windows版本（例如，家庭版或专业版）以及语言和体系结构。取消勾选对这台电脑使用推荐的选项，则可以根据自身需求选择相应的语言和版本。请根据自身诉求选择是否取消勾选，然后点击下一页。


在选择要使用的介质页，选择 U 盘，点击下一页。


在选择 U 盘页，出现 U 盘的盘符（如 E: 无标签），表明 U 盘已连接到系统，点击下一页。



如果出现找不到 U 盘的提示，则表明系统未检测到 U 盘，请确认 U 盘已经连接到系统，然后点击刷新驱动器列表。

计算机将自动下载 Windows，下载完成后，点击下一页。


U 盘制作完成后，提示你的 U 盘已准备就绪页，点击完成，此时 U 盘可以用来安装系统。


## 2. 通过系统安装盘安装 Windows11 系统


安装前您可以将重要文件备份至外部存储器，以免丢失您的重要文件。

将前面制作好的 U 盘，插在待重装系统的计算机上。在计算机开机时，长按或点按 F12 键，然后选择 U 盘启动，按 Enter 键。


选择要安装的语言、时间和货币格式、键盘和输入方法，点击下一步。


系统会自动进入安装界面，点击现在安装。


在适用的声明和许可条款的界面中，用户阅读后若同意则勾选我接受 Microsoft 软件许可条款，点击下一页。


提示你想将 Windows 安装在哪里，选择驱动器 0 分区3：Windows，点击下一页 。

如果电脑磁盘分区已经彻底删除，请先选中磁盘，再点击下图中的新建新建主分区。之后点击格式化，选择文件类型为 NTFS，再根据界面提示进行后续操作。



将会出现界面提示：计算机正在安装 Windows。


在系统安装完成后，计算机会自动重启，此时请拔掉U盘，以防止再次进入安装界面。
重启后会进入到首次开机界面，表示计算机已重装系统完成。按界面提示，完成首次开机系统配置。


## 3. 下载安装网卡驱动后连接网络，下载华为电脑管家，安装相应驱动

使用预先准备的可联网计算机，下载通用的网卡驱动：https://www.intel.cn/content/www/cn/zh/download/19351/intel-wireless-wi-fi-drivers-for-windows-10-and-windows-11.html
将下载好的网卡驱动通过U盘拷贝至重装好系统的计算机上，双击驱动.exe可执行文件，根据提示安装完成后重启计算机即可。
重装后的计算机未安装驱动，许多功能将无法使用。请连接网络后，前往华为消费者业务官网下载华为电脑管家。
安装好华为电脑管家后，打开管家应用，点击电脑优化＞驱动管理＞检查更新，在网络畅通的情况下选择一键升级，管家将自动升级安装您计算机所需的相应驱动。
驱动升级完成后，重启计算机，您可以在设备管理器和管家中看到驱动安装完毕。
